# ГраффитАнимация

Полнофункциональный редактор анимаций для Windows Forms на C#.

## Возможности

### 🎨 Инструменты рисования
- **Кисть** - рисование с регулируемым размером
- **Ластик** - стирание нарисованного
- **Заливка** - заполнение области цветом (flood fill)
- **Выбор цвета** - полная палитра цветов

### 📐 Система слоев
- Множество слоев на один кадр
- Управление видимостью слоев
- Изменение порядка слоев (Z-индекс)
- Регулировка прозрачности

### 🎬 Система кадров
- Неограниченное количество кадров
- Добавление и удаление кадров
- Переключение между кадрами
- Отображение позиции в проекте

### ⏱️ Управление анимацией
- Изменение FPS (1-120 кадров в секунду)
- Встроенный плеер для просмотра
- Плавное воспроизведение с правильной скоростью

### 💾 Сохранение и загрузка
- Собственный формат **.grfan** (ZIP архив)
- Быстрое сохранение проектов
- Загрузка ранее созданных проектов

### 📤 Экспорт
- **GIF** - экспорт анимации в формат GIF
- **Видео** - экспорт в MP4/AVI с кодеком H.264
- **Музыка** - поддержка добавления аудиодорожек к видео

### 🎵 Музыка
- Добавление MP3 и WAV файлов
- Автоматическая синхронизация при экспорте
- Встраивание музыки в финальное видео

## Требования

- **.NET 6.0** или выше
- **Windows 10/11**
- **FFmpeg** (для экспорта в GIF и видео)
  - Скачайте с https://ffmpeg.org/download.html
  - Добавьте папку ffmpeg в PATH или укажите путь в приложении

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/RonjaFox/GraffitAnimation.git
cd GraffitAnimation
```

2. Восстановите зависимости:
```bash
dotnet restore
```

3. Запустите приложение:
```bash
dotnet run
```

## Использование

### Создание новой анимации
1. Перейдите в меню **Файл → Новый проект**
2. Укажите название, ширину и высоту холста
3. Нажмите OK

### Рисование
1. Выберите инструмент на левой панели (Кисть, Ластик, Заливка)
2. Отрегулируйте размер кисти ползунком
3. Выберите цвет кнопкой "Цвет"
4. Рисуйте на холсте

### Работа со слоями
1. Нажмите "+ Добавить слой" для создания нового слоя
2. Рисуйте на активном слое
3. Переключайтесь между слоями в списке

### Создание анимации
1. Нарисуйте первый кадр
2. Нажмите "+ Новый кадр"
3. Нарисуйте следующий кадр
4. Повторите для всех кадров

### Просмотр анимации
1. Установите желаемое значение FPS
2. Нажмите кнопку "▶️ Воспроизведение"
3. Нажмите "⏹️ Стоп" для остановки

### Сохранение проекта
1. Меню **Файл → Сохранить**
2. Выберите папку и имя файла
3. Файл сохранится в формате .grfan

### Экспорт анимации

#### В GIF:
1. Меню **Файл → Экспорт в GIF**
2. Выберите путь сохранения
3. Дождитесь завершения экспорта

#### В видео:
1. (Опционально) Меню **Музыка → Добавить музыку**
2. Меню **Файл → Экспорт в видео**
3. Выберите путь сохранения
4. Дождитесь завершения экспорта

## Горячие клавиши

- **Ctrl+S** - Сохранить проект
- **Ctrl+O** - Открыть проект
- **Ctrl+E** - Экспорт в GIF

## Решение проблем

### FFmpeg не найден
- Убедитесь, что FFmpeg установлен и добавлен в PATH
- Или измените переменную `ffmpegPath` в файле `ExportService.cs`

### Экспорт занимает долгое время
- Это нормально для больших проектов
- Время зависит от количества кадров и разрешения

### Проблемы с музыкой
- Используйте стандартные форматы: MP3, WAV
- Проверьте, что файл не повреждён
